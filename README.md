# 개요

- 스프링 어플리케이션 내에서 외부의 REST API를 다양한 형태로 호출하는 방법을 학습 
- JSON/XML 결과를 받아, 이를 파싱하는 방법을 학습

# 다룰 내용

- GET 방식의 호출
    - Parameter Building
    - URI 인코딩
- POST 방식의 호출
    - Request Body 설정
- Request Header
    - Accpet 설정
- Request Body
    - JSON 형식의 Request Body 전송
- Response Header
    - HttpStatus 코드 검증
- Response Body
    - JSON 파싱
    - XML 파싱

# 스택

- Spring boot 2.3.12
- Spring Web
- Spring Web Services
- H2
- Spring REST Docs
